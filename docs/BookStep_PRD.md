# **BookStep\_PRD**

## **1\. 서비스 정보**

* 서비스명 : BookStep  
* 팀명 : StepUp  
* 플랫폼 : 반응형 웹(모바일 우선)

---

## **2\. 핵심 사용자**

* 문학 도서에 관심 있는 중·고등학생

---

## **3\. 사용자 문제**

* 어떤 책을 읽어야 할지 모르겠다.  
* 검색 결과가 많아 원하는 도서를 빠르게 찾기 어렵다.  
* 관심 도서를 따로 관리하기 불편하다.

---

## **4\. 사용자 니즈**

* 자신에게 맞는 도서를 추천받고 싶다.  
* 원하는 책을 빠르게 검색하고 싶다.  
* 관심 도서를 저장하고 관리하고 싶다.  
* 대출·예약을 간편하게 이용하고 싶다.

---

## **5\. 서비스 목표**

* 사용자가 관심 장르 기반의 추천을 받고, 도서를 검색·확인한 뒤 북마크 및 대출·예약까지 하나의 서비스에서 경험할 수 있도록 한다.

---

## **6\. 화면 구성**

### **1\) 로그인/회원가입 화면**

**기능**

* 회원가입

LocalStorage를 활용하여 이메일·비밀번호 등 정보를 입력해 계정을 생성한다.

* 로그인

LocalStorage를 활용하여 가입한 계정으로 서비스에 로그인한다.

* 관심 장르 선택

가입 시 여러 장르를 선택하여 맞춤 추천 기반을 생성한다.

### **2\) 메인(홈) 화면**

**기능**

* 인기 도서

10대에게 인기 있는 도서를 설문 기반으로 추천 및 큐레이션하여 제공한다.

### **3\) 검색 화면**

**기능**

* 도서 검색

제목·저자·장르를 기준으로 도서를 자연어 및 다국어 검색할 수 있다.

* 필터·정렬

인기순, 최신순 등으로 검색 결과를 정렬 및 필터링할 수 있다.

### **4\) 도서 상세 화면**

**기능**

* 도서 줄거리 조회

선택한 도서의 줄거리와 기본 정보를 제공한다.

* 대출·예약

대출 가능 여부를 확인하고 대출 및 예약 기능을 제공한다.

### **5\) 북마크 화면**

**기능**

* 북마크 목록 조회

북마크한 도서 목록을 확인할 수 있다.

### **6\) 마이페이지**

**기능**

* 개인정보 관리

프로필 조회, 관심 장르 수정, 독서 이력 관리를 제공한다.

* 로그아웃

현재 로그인된 계정에서 로그아웃할 수 있다.

---

## **7\. 사용자 흐름**

회원가입(관심 장르 선택)  
→ 로그인  
→ 메인(홈) 화면  
→ 검색 화면  
→ 도서 상세 화면  
→ 북마크 또는 대출·예약  
→ 마이페이지

---

## **8\. 기술 스택**

* HTML  
* CSS  
* JavaScript  
* LocalStorage
